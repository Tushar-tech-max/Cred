# Cred
This project simulates a real-world credentialing operations dashboard used in healthcare BPO/RCM companies for performance tracking for employess and assigned task 


A real-time dashboard built using Python, SQL (SQLite), and Streamlit with help of AI to track and analyze credentialing case performance in a healthcare workflow.

---

## 🧠 Problem Statement
Credentialing teams face difficulty in tracking:
- Case - ( Task ) processing time
- Pending workload per employee
- Daily productivity

This project solves it by providing a live performance dashboard.

---

## ⚙️ Features

- 📊 Live dashboard using Streamlit
- 🗄️ SQLite database integration
- ⏱️ Case creation & completion timestamp tracking
- 📈 Average processing time calculation
- 👨‍💼 Pending cases per employee view
- 📅 Daily productivity tracking
- 🔄 Auto-refresh support



 Key Metrics Tracked
Cases=Task processed today , Pending cases per employee , Average processing time per case , Completed vs pending ratio
